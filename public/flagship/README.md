# Flagship card media

Drop project media here and the cards on Home + Projects light up automatically.

**Easiest path:** record a ~5–8s screen capture of the live site (or export a clip from
your image/video model), drop the raw file in this folder or anywhere in the Roles folder,
and I'll transcode it to optimized web-ready `<slug>.mp4` + `<slug>.webm` + a `<slug>.png`
poster and wire it into `src/data/projects.ts`.

Slugs: `lupi`, `createsuite`, `alexwelcing`, `high-era`, `lbr-workspace`.

Target specs (I'll produce these): 16:9, ~960px wide, muted, silent, loopable, < ~1.5 MB each.
