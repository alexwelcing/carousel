## Comfy UI Integration: What You'll Build

### Workflow 1: Company Logo + Branding
**Input:** Company name, brand colors, AI focus  
**Output:** Animated wordmark + accent-colored background loop

**Comfy nodes:**
- Text → font renderer (Inter, bold)
- Color gradient → background
- Particle system or glow effect (brand-themed)
- Frame export → PNG/MP4 loop

**Output location:** `assets/generated/[company]-wordmark.mp4`  
**Wire to:** `compositionSchema.ts` → `backgroundAsset` variable

### Workflow 2: Proof Point Previews
**Input:** Project name (Lupi.Live, CreateSuite, alexwelcing.com)  
**Output:** 300×200px preview card image

**Comfy nodes:**
- Screenshot capture (of your existing portfolio sites)
- OR AI-generated abstract preview (based on project description)
- Card frame with title + stat
- Frame export → PNG

**Output location:** `assets/generated/proof-points/[project]-preview.png`  
**Wire to:** `compositionSchema.ts` → `proofPointImages` object

### Workflow 3: Role-Specific Background
**Input:** Company AI focus, role type, fit tier  
**Output:** 1920×1080 animated background (30 seconds)

**Comfy nodes:**
- Noise generator (Perlin, cloud, or custom)
- AI image generator (SDXL or Flux) with role-specific prompt
- Color grading (to brand accent colors)
- Animation keyframes (slow drift, particle systems)
- Frame sequence export → MP4

**Output location:** `assets/generated/backgrounds/[company]-[role]-bg.mp4`  
**Wire to:** `compositionSchema.ts` → `backgroundAsset` variable

### Workflow 4: TTS Voiceover (Phase 2)
**Input:** CTA message + company tone  
**Output:** MP3 audio file + caption SRT

**External API:**
- HeyGen TTS API (or ElevenLabs)
- Input: `compositionSchema.ts` → `ctaText`
- Output: MP3 + word-timing JSON

**Output location:** `assets/generated/voiceovers/[company]-[role]-vo.mp3`  
**Wire to:** `compositionSchema.ts` → `voiceoverAudioUrl` variable
